# dev3_fullstack

Полностью рабочее приложение.

Этот разработчик связал фронтенд и бэкенд, чтобы приложение работало как единое целое.

## План тестирования

1. Подготовка
   - Перейти в папку `dev3_fullstack/Server/src`.
   - Установить зависимости: `npm install`.

2. Автоматические тесты
   - Запустить весь набор Jest: `npm test`.
   - Для постоянного наблюдения за изменениями: `npm run test:watch`.

3. Проверка работы локализации
   - Запустить сервер: `npm run dev`.
   - Открыть страницу `dev3_fullstack/Server/public/index.html` в браузере.
   - Нажать кнопку `EN` / `RU` и убедиться, что элементы интерфейса переводятся.
   - Проверить, что язык сохраняется при перезагрузке.

4. Проверка отзывов
   - На странице товара нажать кнопку `Отзывы`.
   - Убедиться, что отображается список отзывов и карточки выглядят мягко, без резких границ.
   - Если пользователь авторизован, проверить добавление отзыва через форму.

5. Ручное тестирование основных сценариев
   - Логин/регистрация пользователя.
   - Просмотр списка товаров.
   - Добавление товара в корзину.
   - Оформление покупки и проверка истории покупок.
   - Проверка кнопки `Продать` и размещения собственного лота.

6. Проверка API
   - Убедиться, что эндпойнты возвращают данные без ошибок:
     - `GET /api/locale`
     - `POST /api/locale`
     - `GET /api/reviews/:productId`
     - `POST /api/reviews`
   - Проверить, что при отправке отзыва он сохраняется и доступен для просмотра.
